### Prerequisites
- Node.js installed
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/STI-TakoYucky/Cuisine-Hero.git
cd cuisine-hero

# Install dependencies
npm install

# Run the development server
```
